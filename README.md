# OpenClaw_K1-Sagittarius-Arm_Demo
Demo for controlling the Unitree K1 arm (Sagittarius arm) with OpenClaw

This demo is developed based on https://github.com/vanstrong12138/OpenClawPi. <br>
The vision understanding based on VLM in OpenClaw is successful, but I have to admit that the pick-and-place experiment was failed. 
It is a demostration for visitors and has strict time limitation. So I finally developed a record-play system.
